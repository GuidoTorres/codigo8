Elementos de los Formularios

Lista de elementos

Nro Nombre del Elemento Descripcion
1 Input Es un campo de entrada de muchos tipos de datos.
2 Textarea Es un campo de entrad de tipo texto, puede tener mas de una fila.
3 Select Lista de opciones para elegir.
4 Button
  • Para crear botones.
5 Label
  • Elemento que crea una etiqueta asociada a un input.
6 Fieldset
  • Es un elemento que agrupa los campos de un fromulario.
7 Legend
  • Representa al titulo de algunas etiquetas.
  • Por lo general acompaña al fielset.

Tipos de input

Listas de Tipos de input

Numero Tipo Descripcion
1 Text
  • Input para ingresa texto plano.
  • Por ejemplo nombres apellidos, direcciones,etc.
2 Email
  • Para ingresar correos electronicos, se coloca tipo email prar que el formulario valide el ingreso de correo electronico.
3 Number
  • Inpur para ingresar numeros enteros o decimales.
4 Date
  • Input para ingresar fechas.
5 Password
  • Input para ingresar contraseñas el navegador cambia automaticamente el texto ingresado.
5 Time
  • Input para ingresar horas, minutos y segundos.
7 Color
  • Input para selccionar un color de una paleta de colores.
8 Checkbox
  • Input para seleccionar o deseleccionar elementos de mas de un grupo.
  • Es posible seleccionar mas de un chekcbox a la vez.
9 Radio
  • Sirve para selccionar un elemento de un radio button.
  • Solo se puede seleccionar uno a la vez.
10 File
  • Input para selccionar archivo local y mandarlo al servidor.
11 Reset
  • Input en forma de un boton.
  • Sirve para limpiar todos los campos del formulario que se encuentra.
12 Submit
  • Input en forma de un boton.
  • Sirve para enviar todos los datos del formulario al servidor.
